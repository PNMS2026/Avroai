📱 STEP 1: INSTALL GITHUB DESKTOP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  🔗 https://desktop.github.com/
  
  • Download and install
  • Sign in with your GitHub account
  • (Create GitHub account if needed)

📤 STEP 2: PUBLISH YOUR CODE TO GITHUB
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  In GitHub Desktop:
  
  1. File → "Add Local Repository"
  2. Browse to: C:\Users\Administrator\Desktop\avroai
  3. Click "Create Repository"
  4. Click "Publish Repository"
  5. Choose public or private
  6. Click "Publish"
  
  ✅ Your code is now on GitHub!

🌐 STEP 3: SIGN UP FOR RENDER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  🔗 https://dashboard.render.com
  
  • Click "Get Started"
  • Sign in with your GitHub account
  • Authorize Render to access GitHub

➕ STEP 4: CREATE WEB SERVICE ON RENDER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  In Render Dashboard:
  
  1. Click "New +" button (top right)
  2. Select "Web Service"
  3. Find and connect your "avroai" repository
  4. Configure:
     • Name: avroai
     • Branch: main
     • Build Command: npm install
     • Start Command: npm start
     • Instance Type: Free
  
  5. Click "Advanced"
  6. Add Environment Variable:
     • Key: MONGODB_URI
     • Value: mongodb+srv://panesh_mondal:grcQXl5TRG3P76qY@avroai.bkvoht8.mongodb.net/avroai?retryWrites=true&w=majority&appName=Avroai

🚀 STEP 5: DEPLOY!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Click "Create Web Service"
  2. Wait 3-5 minutes for deployment
  3. Watch the logs (exciting!)
  4. When you see "Deploy succeeded" - YOU'RE LIVE!

  Before deploying, make sure MongoDB allows connections:
  
  1. Go to: https://cloud.mongodb.com
  2. Click "Network Access" (left sidebar)
  3. Click "Add IP Address"
  4. Click "Allow Access from Anywhere"
  5. This adds: 0.0.0.0/0
  
  ✅ This allows Render to connect to your database
