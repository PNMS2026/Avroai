   📱 METHOD 1: GITHUB DESKTOP (RECOMMENDED - NO COMMAND LINE!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Download GitHub Desktop
┌────────────────────────────────────────────────────────────────┐
│ The browser should have opened to:                             │
│ https://desktop.github.com/                                    │
│                                                                │
│ ➜ Click "Download for Windows"                                │
│ ➜ Wait for download to complete                               │
│ ➜ Run the installer (GitHubDesktopSetup.exe)                  │
└────────────────────────────────────────────────────────────────┘

STEP 2: Sign In
┌────────────────────────────────────────────────────────────────┐
│ When GitHub Desktop opens:                                     │
│                                                                │
│ ➜ Click "Sign in to GitHub.com"                               │
│ ➜ Browser will open - sign in with:                           │
│    Username: PNMS2026                                          │
│    Password: Your GitHub password                             │
│ ➜ Click "Authorize desktop"                                   │
│ ➜ Return to GitHub Desktop                                    │
└────────────────────────────────────────────────────────────────┘

STEP 3: Add Your Repository
┌────────────────────────────────────────────────────────────────┐
│ In GitHub Desktop:                                             │
│                                                                │
│ ➜ Click "File" menu → "Add local repository"                  │
│ ➜ Click "Choose..." button                                    │
│ ➜ Navigate to: C:\Users\Administrator\Desktop\avroai          │
│ ➜ Click "Add Repository"                                      │
│                                                                │
│ If you see "This directory does not appear to be a Git repo": │
│ ➜ Click "create a repository" link                            │
│ ➜ Uncheck "Initialize this repository with a README"          │
│ ➜ Click "Create Repository"                                   │
└────────────────────────────────────────────────────────────────┘

STEP 4: Commit Your Files
┌────────────────────────────────────────────────────────────────┐
│ You'll see all your project files listed on the left          │
│                                                                │
│ ➜ At the bottom left, you'll see:                             │
│    "Summary (required)" textbox                               │
│ ➜ Type: first commit                                          │
│ ➜ Click the blue "Commit to main" button                      │
│                                                                │
│ ✅ Your files are now committed locally!                       │
└────────────────────────────────────────────────────────────────┘

STEP 5: Publish to GitHub
┌────────────────────────────────────────────────────────────────┐
│ Now you need to push to your GitHub repository:               │
│                                                                │
│ ➜ Click "Publish repository" button (at top)                  │
│                                                                │
│ OR if that's not visible:                                     │
│ ➜ Click "Repository" menu → "Repository settings"             │
│ ➜ In "Primary remote repository (origin)"                     │
│ ➜ Paste: https://github.com/PNMS2026/Avroai.git              │
│ ➜ Click "Save"                                                │
│ ➜ Then click "Push origin" button                             │
└────────────────────────────────────────────────────────────────┘

✅ DONE! Your code is now on GitHub!
   Visit: https://github.com/PNMS2026/Avroai

   💻 METHOD 2: COMMAND LINE (If you prefer Git commands)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Install Git
┌────────────────────────────────────────────────────────────────┐
│ Download Git for Windows:                                      │
│ https://git-scm.com/download/win                              │
│                                                                │
│ ➜ Download and run installer                                  │
│ ➜ Use all default settings                                    │
│ ➜ Click through installation                                  │
│ ➜ Close and reopen PowerShell/Terminal after installation     │
└────────────────────────────────────────────────────────────────┘

STEP 2: Configure Git (First Time Only)
┌────────────────────────────────────────────────────────────────┐
│ Open NEW PowerShell and run:                                   │
│                                                                │
│ git config --global user.name "PNMS2026"                       │
│ git config --global user.email "your-email@example.com"       │
└────────────────────────────────────────────────────────────────┘

STEP 3: Run Your Commands
┌────────────────────────────────────────────────────────────────┐
│ cd C:\Users\Administrator\Desktop\avroai                       │
│                                                                │
│ git init                                                       │
│ git add .                                                      │
│ git commit -m "first commit"                                   │
│ git branch -M main                                             │
│ git remote add origin https://github.com/PNMS2026/Avroai.git  │
│ git push -u origin main                                        │
│                                                                │
│ When prompted for credentials:                                │
│ - Username: PNMS2026                                           │
│ - Password: Use GitHub Personal Access Token                  │
│   (Not your account password!)                                │
└────────────────────────────────────────────────────────────────┘

STEP 4: Create Personal Access Token (If Needed)
┌────────────────────────────────────────────────────────────────┐
│ If Git asks for password, you need a token:                   │
│                                                                │
│ ➜ Go to: https://github.com/settings/tokens                   │
│ ➜ Click "Generate new token" → "Classic"                      │
│ ➜ Name: AvroAI Deploy                                         │
│ ➜ Check: ☑ repo (full control)                               │
│ ➜ Click "Generate token"                                      │
│ ➜ COPY THE TOKEN IMMEDIATELY!                                 │
│ ➜ Use this token as password when pushing                     │
└────────────────────────────────────────────────────────────────┘
